# Planet Microbe Search App

## Backend 

Quick Start:
```
cd backend
npm install
$(npm bin)/nodemon
```

## Frontend

Quick Start:
```
cd frontend
npm install
elm package install
$(npm bin)/elm-live src/Main --output=elm.js --open
  *or*
node_modules/elm/bin/elm make --output=elm.js src/Main.elm
```
